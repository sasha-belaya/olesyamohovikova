# Идеи дизайна: Олеся Моховикова — Сайт-визитка

## Концепция 1: Editorial Noir
<response>
<text>
**Design Movement:** High-Fashion Editorial / Bauhaus Minimalism

**Core Principles:**
- Жёсткая асимметрия: текст и изображение никогда не центрированы симметрично
- Типографика как главный визуальный элемент — крупные заголовки занимают пространство
- Монохром с единственным акцентом: глубокий карминовый (#8B0000)
- Пространство «дышит» — широкие поля, воздух между блоками

**Color Philosophy:**
- Фон: чистый белый #FFFFFF и глубокий чёрный #0A0A0A
- Текст: чёрный на белом / белый на чёрном
- Акцент: карминово-красный oklch(0.38 0.18 25) — только для ключевых элементов (линии, числа, кнопки)
- Серый: oklch(0.55 0 0) для вторичного текста

**Layout Paradigm:**
- Асимметричная сетка: Hero — изображение занимает правую 40%, текст — левую 60%
- Горизонтальные разделители — тонкие линии 1px карминового цвета
- Секции чередуются: белый фон / чёрный фон
- Числа-результаты — огромный шрифт, почти декоративный

**Signature Elements:**
1. Тонкая вертикальная красная линия слева от заголовков секций
2. Крупные номера секций (01, 02...) как декоративный элемент
3. Текст с трекингом (letter-spacing) для заголовков

**Interaction Philosophy:**
- Hover на карточках экспертизы: тонкая красная рамка появляется
- Кнопка «Связаться»: инвертируется (чёрный→белый или наоборот)
- Плавный скролл между секциями

**Animation:**
- Fade-in + slide-up при появлении секций в viewport
- Числа «считаются» при появлении (counter animation)
- Минимальные переходы 300ms ease-out

**Typography System:**
- Заголовки: Playfair Display (serif, элегантность)
- Тело: DM Sans (гротеск, читаемость)
- Акцентные числа: Bebas Neue (конденсированный, мощь)
</text>

## Концепция 2: Architectural Precision
<response>
<text>
**Design Movement:** Swiss International Style / Corporate Modernism

**Core Principles:**
- Сетка как закон: всё выровнено по строгой модульной сетке
- Информационная иерархия через размер и вес шрифта
- Контраст как коммуникация: чёрное и белое говорят сами за себя
- Красный — только как сигнал, не декор

**Color Philosophy:**
- Белый фон как «бумага»
- Чёрный текст как «чернила»
- Красный oklch(0.45 0.22 27) — для акцентов и CTA
- Светло-серый oklch(0.95 0 0) для фоновых секций

**Layout Paradigm:**
- Строгая 12-колоночная сетка
- Hero: полноэкранный с фото на всю высоту справа
- Горизонтальные полосы чередующихся фонов
- Таблицеобразная структура для опыта

**Signature Elements:**
1. Красные горизонтальные линии-разделители
2. Жирные засечные заголовки
3. Монопространственный шрифт для цифр

**Interaction Philosophy:**
- Строгие, предсказуемые переходы
- Hover-эффекты: подчёркивание красным

**Animation:**
- Минимальные, деловые анимации
- Slide-in слева для текста

**Typography System:**
- Заголовки: Cormorant Garamond
- Тело: IBM Plex Sans
- Цифры: IBM Plex Mono
</text>

## Концепция 3: Luxury Magazine — ВЫБРАННАЯ
<response>
<text>
**Design Movement:** Luxury Print Magazine / Vogue Editorial Aesthetic

**Core Principles:**
- Элегантность через контраст масштабов: огромное и маленькое рядом
- Фотография как главный герой — большая, смелая, без рамок
- Красный как «подпись» — редкий, но незабываемый
- Движение через типографику: разные веса создают ритм

**Color Philosophy:**
- Доминирует белый: oklch(0.99 0 0) — чистота и пространство
- Чёрный: oklch(0.08 0 0) — глубина и авторитет
- Благородный красный: oklch(0.42 0.20 26) — #9B1C1C — тёплый, не кричащий
- Тёплый серый: oklch(0.60 0.005 60) — для вторичных элементов

**Layout Paradigm:**
- Hero: фото Олеси занимает правую половину экрана, текст — левая, с большим отступом сверху
- Нет симметрии — каждая секция имеет свой уникальный layout
- Секция результатов: 4 огромные цифры на тёмном фоне
- Экспертиза: асимметричная плитка разных размеров

**Signature Elements:**
1. Тонкая красная горизонтальная черта над заголовками секций (декоративный rule)
2. Крупные serif-заголовки с тонким трекингом
3. Чёрные секции с белым текстом чередуются с белыми

**Interaction Philosophy:**
- Кнопки с hover-инверсией: белая→чёрная с красной рамкой
- Карточки экспертизы: появление красной линии снизу при hover
- Плавный параллакс на фото в Hero

**Animation:**
- Framer Motion: staggered fade-in для элементов
- Числа считаются при скролле (IntersectionObserver)
- Smooth scroll по всему сайту

**Typography System:**
- Заголовки: Cormorant Garamond (serif, роскошь, элегантность)
- Подзаголовки: Cormorant Garamond Italic
- Тело текста: Jost (гуманистический гротеск, читаемость)
- Акцентные числа: Cormorant Garamond Bold — огромный размер
- Трекинг заголовков: 0.05em–0.15em для воздушности
</text>

## Выбранная концепция: Luxury Magazine (#3)

Этот подход наилучшим образом отражает профессиональный уровень Олеси:
- Cormorant Garamond создаёт ощущение премиального издания
- Асимметричные layouts делают сайт запоминающимся
- Красный используется точечно — как подпись, а не декор
- Чёрно-белые секции создают визуальный ритм и не дают скучать
